\chapter{Licencias}
\label{apendice_licencias}

Este apéndice recoge la información relativa a licencias y uso de recursos externos empleados durante el desarrollo del proyecto.

\section{Código fuente}

El código fuente desarrollado para este Trabajo de Fin de Grado tiene finalidad académica. Incluye tanto el backend en Spring Boot como el frontend en React, así como los archivos propios de configuración y documentación generados durante el desarrollo.

En caso de publicarse o reutilizarse el proyecto fuera del contexto académico, debería añadirse un archivo de licencia específico en el repositorio, indicando con claridad los permisos de uso, copia, modificación y distribución.

\section{Dependencias software}

El proyecto utiliza dependencias externas gestionadas mediante Gradle y npm. Entre ellas se encuentran Spring Boot, Spring Security, Spring Data JPA, H2, React y otras bibliotecas auxiliares necesarias para la ejecución y prueba de la aplicación.

Cada una de estas dependencias conserva su licencia original. La instalación mediante los gestores de dependencias no modifica dichas licencias, por lo que cualquier redistribución del proyecto debería respetar las condiciones establecidas por cada biblioteca.

\section{Recursos gráficos y fuentes externas}

La aplicación utiliza recursos gráficos, imágenes de referencia e información procedente de páginas externas. Estas fuentes se recogen en la bibliografía de la memoria, incluyendo la fecha de consulta correspondiente.

Las imágenes utilizadas con finalidad ilustrativa pertenecen a sus respectivos autores o plataformas de origen. En este trabajo se emplean dentro de un contexto académico y documental.

\section{Servicios externos}

El inicio de sesión se apoya en Google OAuth 2.0. El uso de este servicio está sujeto a las condiciones establecidas por Google para sus APIs y servicios de autenticación.

Asimismo, la aplicación puede enlazar o apoyarse en documentación externa, como artículos técnicos, recursos históricos o páginas de referencia. Cada una de estas fuentes mantiene sus propias condiciones de uso.
